# Frontend

Aplicacion PWA del chat generico.

## Responsabilidad

- UI del chat.
- Servicios frontales.
- Contratos de transporte REST y gRPC/gRPC-web.
- Configuracion visual.
- PWA y persistencia local.

## Stack

- Pug
- Sass
- Vite
- vite-route
- TypeScript
- vitest (testing)
- ESLint + Prettier (quality)

## Comandos

Ejecutar desde la raiz del proyecto:

```bash
just dev          # Dev server (http://localhost:5173)
just dev-mock     # Dev server + mock backend
just test         # Tests
just lint         # Linter
just build        # Production build
```
